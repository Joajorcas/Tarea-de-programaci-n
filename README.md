# 🐉 Dragones & Castillos - Página Web

Una página web moderna y responsiva sobre un juego épico de fantasía con dragones y castillos medievales.

## 🎨 Características

- **Diseño Moderno**: Colores nocturnos (azules oscuros, purpuras, dorados)
- **Animaciones Suaves**: Efectos hover, scroll animations y transiciones elegantes
- **Completamente Responsivo**: Se adapta perfectamente a cualquier dispositivo (móvil, tablet, desktop)
- **Múltiples Secciones**:
  - Hero section impactante
  - Sección de historia
  - Mecánicas del juego
  - Barrios del reino
  - Galería de imágenes
  - Call-to-action para descarga
  - Footer con links

## 🚀 Despliegue en Vercel

### Opción 1: Git + Vercel Dashboard (Recomendado)

1. **Sube el proyecto a GitHub:**
   - Crea un repositorio en GitHub
   - Haz git push con tu código

2. **Deploy en Vercel:**
   - Ve a [vercel.com](https://vercel.com)
   - Inicia sesión con tu cuenta GitHub
   - Haz clic en "New Project"
   - Selecciona tu repositorio
   - Vercel detectará automáticamente que es un sitio estático
   - Haz clic en "Deploy"

### Opción 2: Vercel CLI

```bash
# Instala Vercel CLI
npm i -g vercel

# En la carpeta del proyecto
vercel

# Sigue las instrucciones interactivas
```

### Opción 3: Drag & Drop (Más rápido)

1. Ve a [vercel.com/new](https://vercel.com/new)
2. Arrastra y suelta tu carpeta del proyecto
3. Vercel iniciará el despliegue automáticamente

## 🔧 Personalización

### Cambiar Colores
Edita las variables CSS en `style.css`:
```css
:root {
    --primary-dark: #0a0e27;
    --secondary-dark: #1a1f3a;
    --accent-purple: #6d28d9;
    --accent-gold: #fbbf24;
    --accent-blue: #3b82f6;
    --text-light: #f1f5f9;
}
```

### Añadir Nuevas Secciones
1. Añade la nueva sección en `Index.html`
2. Crea los estilos en `style.css`
3. Añade la interactividad en `script.js` si es necesaria
4. Actualiza el menú de navegación
